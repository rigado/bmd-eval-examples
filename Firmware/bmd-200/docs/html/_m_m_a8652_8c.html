Detailed Description

Module for assisting in I2C communications with the MMA8652 accelerometer.

This module is capable of performing chip specific operations such as setup. This module does not implement every feature of the MMA8652, but does provide a framework to implement additional functionality. For more details on custom setup parameters, see the datasheet for the MMA8652.
