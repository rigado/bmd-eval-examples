Overview

This class provide control of connections to and disconnections from discovery CBPeripheral devices.

Connection to a device is performed using RigAvailableDeviceData objects. Disconnections are performed using RigLeBaseDevice objects. Additionally, a delegate is provided to alert calling objects when certain events have been performed (e.g. connection/disconnection, connection timeout, etc).
