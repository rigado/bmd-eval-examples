Conforms to NSObject

Overview

@protocol RigLeBaseDeviceDelegate

This protocol provides a delegate interface for notifying when certain device events occur.

– discoveryDidCompleteForDevice: required method

@method discoveryDidCompleteForDevice:

- (void)discoveryDidCompleteForDevice:(RigLeBaseDevice *)device

Parameters

device

The RigLeBaseDevice object for which discovery has finished.

Discussion

This method is called when the discovery of all services and characteristics has completed. After this discovery is complete, all characteristic property values will have been read.

This method is called when all services and characteristics have been discovered for a given device.
