Detailed Description

PWM module for use with the BMD-200 eval board.

This module implements the PWM on the BMD-200 module and provides an example on implementing PWM for the RGB LED on on the BMD-200 EVAL Board. This file contains the code necessary to independently control pwm output on three separate channnels given a 3 byte standard hex color value.
